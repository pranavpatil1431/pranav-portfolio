# Pranav Patil — Personal Portfolio

This repository contains Pranav Patil's personal portfolio website built with React + TypeScript and Vite. It showcases projects, skills, timeline, certificates, achievements, and contact information.

Local development:

```bash
cd Pranav_Portfolio-main
npm install
npm run dev
# open http://localhost:5173
```

Build for production:

```bash
cd Pranav_Portfolio-main
npm run build
```
